Merge two single-sided Prns into one, double-sided Prn.

Args

front
A single-sided Prn that will become the front of the resultant, double-sided Prn.
rear
A single-sided Prn that will become the back of the resultant, double-sided Prn.
front_as_base
Whether or not to use fronts 'card-options' element as the base, with 'card-options' nodes taken from rear only if they do not exist in front. If False then rears 'card-options' element is taken as the base.

Returns

The resultant double-sided Prn.

Raises

PrnParseError: If an error occured when parsing either front or rear, or when creating the resultant Prn.

class Prn (command: Union[bytes, xml.etree.ElementTree.Element] = None, payload: bytes = b'', raw_header: bytes = None)

A class that handles construction, parsing and manipulation of Magicard's Generation 2 printer Prn files.

Initialise a Prn object, either as an empty Prn or with a pre-defined XML header and/ or payload.

Args

command
If provided, then this is inserted into the XML header of the Prn object. Note that if raw_bytes is provided then this is ignored.
payload
Used to setup the payload. If None then the Prn object will have an empty payload.
raw_header
If provided, then the header is parsed from this. Note that this takes precedence over command.
