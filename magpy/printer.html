Module magpy.printer

class Printer (conn: Connection)

def wait_for_printer(self, timeout: int) ‑> NoneType

Wait for the printer to become idle.

This method will not return until the printer becomes idle, i.e. until it is not performing any of the following:

  • Executing a print job.
  • Executing a Printer.card_move(), Printer.mag_read(), Printer.mag_read_raw(), or a Printer.set() method.
  • Reinitialising the print engine, e.g while clearing an error.
  • Winding the rollers.
  • Performing a cleaning cycle.
  • Performing a soak routine.
  • Updating the system software.
  • Performing a specialising Engineering test routine, reserved for internal use.

Args

timeout
The time to wait, in ms, for the printer to become idle.

Raises

ConnectionError: If an error occured during comms with the printer.

PrnParseError: If an unexpected response is received from the printer.
